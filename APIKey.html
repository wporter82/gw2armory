API Key

In order to use this site, you will need to generate an API key with all permissions for your account and enter it here. The permissions are read-only. More info about this process can be found on the GW2 Wiki.

Security Notice
Your API key is stored locally on your computer and not saved to the server to reduce the risk of security vulnerabilities. This site promises to not use your API key for anything other than to show you information about your account. We do not have access to your password and the Guild Wars 2 API is read-only access. If the API were to change in the future, we promise to uphold these statements.
